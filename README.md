# My Portfolio (Migrate Application)

## Stack: 

- TypeScript
- React
- Html
- TailWind CSS
- Styled Components
- Next.js
- Email JS
- Docker
- Docker-Compose
- Lib Shadcn UI 

### How to install:

- Git Clone
- With docker and docker-compose installed, the following command moves up the stack:
- make up and open:
- localhost:3000
- Clear the stack:
- make clean
- For the stack:
- make down
